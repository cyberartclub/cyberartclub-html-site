Festival: An Interactive Dream
is a small, text-based interactive game based on hazy dreams and memories of peace. I created this for both my Interactive Design course and personal interest.

I already knew HTML, CSS, and a little bit of Javascript, but working in Harlowe's JSON-based language was a very new experience. I'm glad that my imagination wasn't too demanding of its capabilities, though!

This was my first interactive fiction game, and I was eager to implement as many ideas as I could to enrich the experience and bring meaning to the whimsical scenarios the player will travel through.

** Includes font options for accessibilty, responsive windows, custom usernames, multiple endings, and a secret level! Click here to log in.

Parallel to creating Festival, I also was tasked with designing a soundscape in Unreal Engine 4. It was my toughest project yet, and I'm pleased with how it turned out. You can view the demonstration here on YouTube. Eventually, I'll add an .exe for anyone to download as well. After completing it, I'm not too happy to admit that I'm adding "game design" to my enormous list of hobbies. On the other hand, I was able to make something positive out of my forgetting-to-cancel-my-free-trial from Epidemic Sound, which cost me... a lot.

I'm surprised at how much I was able to accomplish in a handful of months - even this new website design! Maybe I'll run a blog, start a podcast...

Honestly, I'm just glad that I was able to get something done worth turning in. I'm officially a bachelor's degree-haver, now, so I feel obligated to use what I've learned to make myself even busier. I think this is a good ending to a very rough start, and a great start to a very unknown.
